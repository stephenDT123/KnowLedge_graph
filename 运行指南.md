# 本科《Python程序设计》教育知识图谱项目 - 运行指南

## 📋 项目概述
本项目为本科《Python程序设计》课程构建基于GraphRAG的教育知识图谱系统，使用DeepSeek LLM和阿里云Embedding模型。

## 🛠️ 环境要求
- **操作系统**: Windows 10/11
- **Python**: 3.9（推荐使用Anaconda py39环境）
- **内存**: 至少4GB
- **网络**: 可访问DeepSeek和阿里云API

## 🚀 快速开始（3分钟）

### 方法一：一键运行（最简单）
1. 打开PowerShell
2. 切换到项目目录：
   ```powershell
   cd "d:\code\Knowledge_Graph"
   ```
3. 运行一键脚本：
   ```powershell
   .\run.ps1
   ```

### 方法二：分步运行

#### 步骤1：安装依赖
```powershell
cd "d:\code\Knowledge_Graph"
& "C:\ProgramData\anaconda3\envs\py39\python.exe" -m pip install PyPDF2 jieba pillow pytesseract pymupdf openai numpy pydantic fastapi uvicorn
```

#### 步骤2：测试API
```powershell
& "C:\ProgramData\anaconda3\envs\py39\python.exe" models/simple_test.py
```
✅ 预期输出：两个API都显示"成功"

#### 步骤3：构建知识图谱
```powershell
& "C:\ProgramData\anaconda3\envs\py39\python.exe" -c "
import asyncio
import sys
sys.path.insert(0, 'models')
from graphrag_builder import build_graphrag_dataset

async def main():
    result = await build_graphrag_dataset(
        output_dir='models/graphrag_output',
        pdf_paths=['models/附件：学员操作手册.pdf']
    )
    print(f'✅ 构建完成：{result[\"documents\"]}个文档，{result[\"entities\"]}个实体，{result[\"relationships\"]}个关系')

asyncio.run(main())
"
```

#### 步骤4：启动服务
```powershell
& "C:\ProgramData\anaconda3\envs\py39\python.exe" -m uvicorn models.main:app --reload --host 0.0.0.0 --port 8000
```

#### 步骤5：访问服务
- 打开浏览器访问：http://localhost:8000
- API文档：http://localhost:8000/docs

## 🔧 配置说明

### API密钥配置
请通过环境变量配置 API 密钥：
- **DeepSeek LLM**: 设置 `DEEPSEEK_API_KEY`
- **阿里云Embedding**: 设置 `ALIYUN_EMBEDDING_API_KEY`

Windows PowerShell 示例：
```powershell
$env:DEEPSEEK_API_KEY="your_deepseek_api_key"
$env:ALIYUN_EMBEDDING_API_KEY="your_aliyun_embedding_api_key"
```

### 使用自己的PDF文件
```python
result = await build_graphrag_dataset(
    output_dir="models/graphrag_output",
    pdf_paths=["您的PDF文件路径.pdf"]  # 替换为您的文件
)
```

## 🐛 故障排除

### 问题1：ModuleNotFoundError
**症状**: `No module named 'openai'` 或类似错误
**解决**:
```powershell
& "C:\ProgramData\anaconda3\envs\py39\python.exe" -m pip install openai
```

### 问题2：API调用失败
**症状**: `DeepSeek API调用失败` 或 `阿里云Embedding调用失败`
**解决**:
1. 检查网络连接
2. 验证API密钥是否正确
3. 运行测试脚本确认：
   ```powershell
   & "C:\ProgramData\anaconda3\envs\py39\python.exe" models/simple_test.py
   ```

### 问题3：PDF处理失败
**症状**: `缺少PDF解析依赖`
**解决**:
```powershell
& "C:\ProgramData\anaconda3\envs\py39\python.exe" -m pip install PyMuPDF PyPDF2
```
